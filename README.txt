1. Implementation of an integer programming for clustering. For more details about the model go to [1]
2. The model was solved using cplex 12.7.1
3. How to run:
	3.1- run the makefile to generate "a.out"
	3.2- ./a.out input_dataset_file number_of_clusters

[1] PIRIM, Harun; EKSIOGLU, Burak; GLOVER, Fred W. A Novel Mixed Integer Linear Programming Model for Clustering Relational Networks. Journal of Optimization Theory and Applications, v. 176, n. 2, p. 492-508, 2018.
